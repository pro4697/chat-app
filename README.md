# Chat App (2021.01)

- https://chat-app-d86d1.web.app

### Stack

- React JS

### Library

- Firebase
- Bootstrap

### etc

- Firebase를 이용한 실시간 채팅 웹앱 구현
- react-hook-form을 이용하여 로그인 / 회원가입 처리
- Function component & Class component
- React Hook, Redux
- Ref를 이용한 돔 접근
- 즐겨찾기, 채팅방 추가, 채팅 검색, DM 보내기, 새 메시지 알림, 이미지 전송, 프로필 사진
